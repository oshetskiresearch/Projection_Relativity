# PR-IV Numerical Scope

PR-IV introduces no fitted parameter. Numerical values quoted from PR-I
through PR-III are regenerated by the public modules under `../symbolic/code`
and written to `../symbolic/results` so the PR-IV paper can be checked as one
self-contained package.

The master runner applies frozen diagnostic values only after each candidate
has been generated. It therefore reproduces the tables without using their
agreement to select `sigma8`, `N_gen`, the projector, or the neutral
coefficient `k`.

Exact symbolic identities use zero tolerance. The explicit floating-point
tolerance and runtime policies are recorded in `../numerical_tolerances.json`
and `../runtime_manifest.json`.
