# PR-III External Alignment Ledger

This ledger tracks outside papers, observations, and theoretical developments that are relevant to PR-III or to the broader PR program.

The purpose is not to collect confirmations. The purpose is to classify external material correctly and prevent hype from contaminating the closure ledger.

---

## Classification key

| Class | Meaning |
|---|---|
| Direct test | Directly tests a PR prediction or failure condition |
| Benchmark analog | Uses a different mechanism but produces comparable macroscopic behavior |
| Adjacent prior art | Conceptually related but structurally distinct |
| Contrast case | Useful because it shows what PR is not |
| Future-study seed | Suggests a possible future PR extension |

## Public-language rule

Use:

> This result is compatible with, motivates, or benchmarks a PR sector.

Avoid:

> This confirms PR.

The PR program gains credibility by surviving external pressure and producing clear failure conditions, not by overclassifying every aligned observation as evidence.
